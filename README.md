# Student Performance Predictor

## Project Overview
This project predicts student final marks using Machine Learning.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Google Colab

## Features
- Data preprocessing
- Student performance prediction
- Model evaluation
- Data visualization

## Model Used
Linear Regression

## Results
- MAE: 0.4398
- MSE: 0.2155
- R² Score: 0.9995
